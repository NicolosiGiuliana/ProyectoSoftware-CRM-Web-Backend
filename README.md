Este repositorio contiene el desarrollo academico de un sistema CRM destinado a una agencia de marketing digital dedicada a la gestión de campañas publicitarias y redes sociales para PYMES.

El objetivo del proyecto es centralizar la administración de clientes, proyectos, tareas e interacciones, permitiendo al equipo de la agencia gestionar campañas de manera eficiente, realizar seguimiento detallado de cada proyecto y facilitar la comunicación con los clientes.

El trabajo se entrega en dos etapas:

Parte 1 – Backend (API REST): Implementación de la lógica de negocios, endpoints, ORM, modelo de datos y criterios de aceptación definidos por la cátedra.
Parte 2 – Frontend: Desarrollo de la interfaz web con HTML, CSS y JavaScript consumiendo la API creada en la Parte 1.

Este repositorio corresponde a la Parte 1 (Backend), mientras que el frontend será subido posteriormente en un repositorio separado.
